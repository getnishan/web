BEFORE DEPLOYMENT - ACTION ITEMS:
================================================================================

1. ⚠️ UPDATE db.php:
   - Change DB_HOST, DB_USER, DB_PASS, DB_NAME
   - Use your actual database credentials

2. ⚠️ CHANGE ADMIN PASSWORD:
   - Edit admin_login.php (line 22)
   - Change from 'admin123' to a secure password

3. ⚠️ CREATE DATABASE:
   - Import database.sql to create the applications table
   - Or run the SQL commands manually

4. ⚠️ SET PERMISSIONS:
   - chmod 755 uploads
   - chmod 755 uploads/videos
   - chmod 644 *.php *.html *.css *.js

5. ⚠️ RUN SETUP SCRIPT:
   - Access setup_directories.php once via browser
   - Or create uploads/videos/ manually

6. ⚠️ CONFIGURE PHP:
   - Set upload_max_filesize (recommended: 50M)
   - Set post_max_size (recommended: 50M)
   - Set max_execution_time (recommended: 300)

7. ⚠️ TEST FUNCTIONALITY:
   - Form submission
   - Video recording and upload
   - Admin login
   - Video playback in admin panel
   - CSV export
   - PDF export
